# pkm consolidate

List daily notes eligible for consolidation.

## Usage
`pkm consolidate [OPTIONS] COMMAND [ARGS]...`

Run without subcommand to list candidates. Use the `mark` subcommand to mark a daily note as consolidated.

## Commands
- **`mark`**: Mark a daily note as consolidated.

## Examples
```bash
pkm consolidate
pkm consolidate mark 2026-04-05
```
